Tema 2

Costos Variables

🗒️ Definición

Los costos variables cambian de acuerdo con la cantidad de productos elaborados o servicios ofrecidos. Son esenciales para calcular precios de venta, ganancias y niveles de producción rentables.

⚙️ Características

  • Cambian según el volumen de producción.
  • Aumentan o disminuyen dependiendo de las ventas.
  • Son proporcionales a la actividad del negocio.
📟 JavaScript
// Plantilla: Costos Variables
        let costoPorUnidad = recuperarDecimal('cvu');     // ej. 3
        let cantidad       = recuperarEntero('cantidad'); // ej. 50

        let totalCV = costoPorUnidad * cantidad;
        mostrarTexto('resultado', "Costo Variable: $" + totalCV);
        // Resultado: Costo Variable: $150

🧮 Calculadora Interactiva

❌ Errores Comunes

  • Confundir costos variables con costos fijos.
  • No actualizar el costo variable unitario cuando cambian los insumos.
  • Calcular el total sin incluir todos los materiales.

💡 Tips

  • Test rápido: ¿Este gasto aumenta si produzco más? Si SÍ → es costo variable.
  • Fórmula directa: CV Total = Costo Unitario × Cantidad producida.
  • Consejo: Negociar precios con proveedores reduce los costos variables directamente.
Tema 4

Costos de Materia Prima

🗒️ Definición

El control adecuado de la materia prima permite reducir desperdicios, mejorar la producción y obtener mayores ganancias dentro de una empresa.

⚙️ Tipos

  • Directa: ingredientes principales del producto.
  • Indirecta: empaques, etiquetas, condimentos.
📟 JavaScript
// Plantilla: Materia Prima
          let harina = recuperarDecimal('harina'); // ej. 2
          let huevos = recuperarDecimal('huevos'); // ej. 3
          let azucar = recuperarDecimal('azucar'); // ej. 1

          let totalMP = harina + huevos + azucar;
          mostrarTexto('resultado', "Costo Materia Prima: $" + totalMP);
          // Resultado: Costo Materia Prima: $6

🧮 Calculadora Interactiva

❌ Errores Comunes

  • No considerar las mermas o desperdicios en el costo de materia prima.
  • Omitir la materia prima indirecta (empaques) al calcular el costo de producción.
  • No actualizar los precios de insumos ante variaciones del mercado.

💡 Tips

  • Factor de merma: Si la receta pide 100g y se desperdicia un 10%, el costo real es sobre 110g.
  • Costeo exacto: Incluye TODOS los insumos: ingredientes, empaques y materiales indirectos.
  • Consejo: Actualiza los costos cada vez que cambien los precios de proveedores.
Tema 6

Costeo de Recetas

🗒️ Definición

El costeo de recetas permite calcular el costo total de los ingredientes de un alimento o bebida. Muy utilizado en restaurantes y cafeterías para fijar precios correctos.

⚙️ Elementos del Costeo

  • Ingredientes y sus cantidades.
  • Precio unitario de cada ingrediente.
  • Costo total por receta.
  • El margen de ganancia se agrega al costo para fijar el precio de venta.
📟 JavaScript
// Costeo de Recetas
        let ingrediente1 = { nombre: "Harina", costo: recuperarDecimal('i1') };
        let ingrediente2 = { nombre: "Huevos", costo: recuperarDecimal('i2') };
        let ingrediente3 = { nombre: "Azúcar", costo: recuperarDecimal('i3') };

        let costoTotal = ingrediente1.costo + ingrediente2.costo + ingrediente3.costo;
        mostrarTexto('resultado', "Costo Total Receta: $" + costoTotal.toFixed(2));
        // Resultado: Costo Total Receta: $4.50

🧮Calculadora interactiva

IngredienteCosto ($)

❌ Errores Comunes

  • No considerar el desperdicio o merma al calcular el costo.
  • Olvidar ingredientes secundarios (sal, condimentos, empaque).
  • No actualizar los precios de ingredientes cuando varían en el mercado.

💡 Tips

  • Formato: Usa una tabla: Ingrediente │ Cantidad │ Precio unitario │ Costo parcial.
  • Regla del costo objetivo: El costo no debe superar el 30–35% del precio de venta.
  • Hack productivo: Pesa y mide con precisión — 10g extra × 100 platos es significativo.
Tema 8

Punto de Equilibrio

🗒️ Definición

El punto de equilibrio permite identificar el mínimo nivel de ventas necesario para cubrir los costos de operación. Fundamental para la planificación financiera.

⚙️ Fórmula

  • Bajo el PE → pérdida
  • En el PE → sin ganancia ni pérdida
  • Sobre el PE → ganancia
📟 JavaScript
// Punto de Equilibrio
        let CF  = recuperarDecimal('cf');   // Costos Fijos ej. 1000
        let PV  = recuperarDecimal('pv');   // Precio de Venta ej. 20
        let CVU = recuperarDecimal('cvu');  // Costo Variable Unitario ej. 10

        let PE = CF / (PV - CVU);
        mostrarTexto('resultado', "Punto Equilibrio: " + PE + " unidades");
        // Resultado: Punto Equilibrio: 100 unidades

🧮 Calculadora Interactiva

❌ Errores Comunes

  • Usar el costo total en lugar del costo variable UNITARIO en la fórmula.
  • No actualizar el punto de equilibrio cuando cambian costos o precios.
  • Confundir punto de equilibrio en unidades con el punto de equilibrio en dinero.

💡 Tips

  • PE en dinero: PE$ = Costos Fijos / (1 − Costo Variable / Precio Venta).
  • Estrategia: Para bajar el PE: reduce costos fijos, sube precio o baja costo variable.
  • Interpretación: Bajo el PE = pérdida. En el PE = cero ganancia. Sobre el PE = ganancia.
