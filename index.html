Tema 1 · Costos Fijos

🗒️ Definición

Los costos fijos representan los gastos permanentes que una empresa debe cubrir para operar. Su correcto control permite mantener estabilidad financiera y planificar adecuadamente los ingresos necesarios para sostener el negocio.

📊 Características

  • No cambian a corto plazo.
  • Se mantienen estables aunque aumente o disminuya la producción.
  • Son permanentes durante un período determinado.

Servicios ($)

📟 JavaScript
let alquiler = 800, internet = 50, seguro = 100;
let totalCF = alquiler + internet + seguro;
// Resultado: Costos Fijos: $950

Tema 2 · Costos Variables

🗒️ Definición

Los costos variables cambian de acuerdo con la cantidad de productos elaborados o servicios ofrecidos. Son esenciales para calcular precios de venta, ganancias y niveles de producción rentables.

📊 Características

  • Cambian según el volumen de producción.
  • Aumentan o disminuyen dependiendo de las ventas.
  • Son proporcionales a la actividad del negocio.

🧮 Calculadora

Costo por Unidad ($)
Cantidad Producida
📟 JavaScript
let costoPorUnidad = 3, cantidad = 50;
let totalCV = costoPorUnidad * cantidad;
// Resultado: Costo Variable: $150

Tema 3 · Costos Directos e Indirectos

🗒️ Definición

La clasificación entre costos directos e indirectos ayuda a organizar mejor los gastos de producción y permite calcular correctamente el costo real de un producto o servicio.

📊 Clasificación

  • Directos: Identificables con un producto específico. Ej: harina, mano de obra en cocina.
  • Indirectos: Necesarios para operar pero no asociados a un solo producto. Ej: luz, arriendo.

🧮 Calculadora

Costos Directos ($)
Costos Indirectos ($)
📟 JavaScript
let harina = 2, manoDeObra = 5;
let costoDirecto = harina + manoDeObra;
let luz = 3, costoIndirecto = luz;
let costoTotal = costoDirecto + costoIndirecto;
// Resultado: Costo Total: $10

Tema 4 · Costos de Materia Prima

🗒️ Definición

El control adecuado de la materia prima permite reducir desperdicios, mejorar la producción y obtener mayores ganancias dentro de una empresa.

📊 Tipos

  • Directa: ingredientes principales del producto.
  • Indirecta: empaques, etiquetas, condimentos.

Tema 5 · Mano de Obra

🗒️ Definición

La mano de obra es fundamental para el funcionamiento de cualquier empresa, ya que influye directamente en la calidad, productividad y eficiencia del proceso productivo.

📊 Clasificación

  • Directa: participa directamente en la fabricación. Ej: chef, operario.
  • Indirecta: apoya el proceso productivo. Ej: contador, guardia de seguridad.

Tema 6 · Costeo de Recetas

🗒️ Definición

El costeo de recetas permite calcular el costo total de los ingredientes de un alimento o bebida. Muy utilizado en restaurantes, cafeterías y negocios gastronómicos para fijar precios correctos.

📊 Elementos del Costeo

  • Ingredientes y sus cantidades.
  • Precio unitario de cada ingrediente.
  • Costo total por receta.
  • El margen de ganancia se agrega sobre el costo total para fijar el precio de venta.

🧮 T6 · Costeo de Recetas

IngredienteCosto ($)
📟 JavaScript
let i1 = { nombre: "Harina", costo: 1.50 };
let i2 = { nombre: "Huevos", costo: 2.00 };
let i3 = { nombre: "Azúcar", costo: 1.00 };
let costoTotal = i1.costo + i2.costo + i3.costo;
// Resultado: Costo Total Receta: $4.50

Tema 7 · Margen de Ganancia

🗒️ Definición

El margen de ganancia permite conocer la rentabilidad de los productos y ayuda a tomar decisiones estratégicas sobre precios y ventas.

📊 Fórmula

  • Ganancia = Precio - Costo
  • % = (Ganancia / Costo) × 100

🧮 Calculadora

Costo Total ($)
Precio de Venta ($)
📟 JavaScript
let costo = 10, precioVenta = 15;
let ganancia = precioVenta - costo;
let porcentaje = (ganancia / costo) * 100;
// Resultado: Ganancia: $5  |  % Ganancia: 50%

Seccion creada por: Oscar Gomez

Tema 8 · Punto de Equilibrio

🗒️ Definición

El punto de equilibrio permite identificar el mínimo nivel de ventas necesario para cubrir los costos de operación. Fundamental para la planificación financiera.

📊 Fórmula

  • Bajo el PE → pérdida
  • En el PE → sin ganancia ni pérdida
  • Sobre el PE → ganancia

🧮 Calculadora

Costos Fijos ($)
Precio de Venta ($)
Costo Variable Unit. ($)
📟 JavaScript
let CF = 1000, PV = 20, CVU = 10;
let PE = CF / (PV - CVU);
// Resultado: Punto Equilibrio: 100 unidades

Evaluación Final · Test de Conocimientos

Responde las 5 preguntas de opción múltiple. Al finalizar recibirás tu calificación con retroalimentación.

01

Una empresa paga $500 de arriendo, $80 de internet y $120 de seguro sin importar cuánto produzca. ¿A cuánto ascienden sus costos fijos mensuales?

✅ Correcto: 500 + 80 + 120 = $700 de costos fijos.
02

El precio de venta de un producto es $25 y su costo de producción es $10. ¿Cuál es el porcentaje de ganancia sobre el costo?

✅ Ganancia = $15. % Ganancia = (15/10) × 100 = 150%.
03

Una pizzería tiene un costo variable por pizza de $3 y produce 50 pizzas. ¿Cuál es el costo variable total?

✅ Costo Variable Total = 3 × 50 = $150.
04

Un chef recibe un salario mensual de $600 y trabaja 160 horas al mes. ¿Cuál es el costo por hora de su mano de obra?

✅ Costo por hora = 600 / 160 = $3.75.
05

Una empresa tiene costos fijos de $2000, precio de venta $40 y costo variable unitario $20. ¿Cuántas unidades debe vender para llegar al punto de equilibrio?

✅ PE = 2000 / (40 - 20) = 2000 / 20 = 100 unidades.
